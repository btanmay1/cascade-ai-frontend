## 🧠 What is Cascade?
Cascade is an AI-powered personal life optimization system that detects bottlenecks in your daily life and suggests the most impactful actions to improve productivity, mental health, and overall performance.

## ✨ Features
- **Cascade Score:** A daily 0-100 score reflecting your overall life optimization.
- **8 Data Source Inputs:** Track Sleep, Work, Money, Health, Relationships, Mental Focus, Mood, and Environment.
- **Lightning-Fast Daily Check-in:** Log all your vitals in under 30 seconds using interactive custom sliders.
- **Bottleneck Detection:** Automatically identifies critical issues impacting your performance.
- **Life Map:** Visualizes cause → effect relationships between your habits and outcomes.
- **Pipeline View:** A clean daily schedule and action timeline.
- **ML Status Tracker:** Monitor the training progress of the background intelligence engine.
- **Premium Design:** Immersive dark UI with glassmorphic cards and purple accent styling.

## 💻 Local Setup

To run this project locally, execute the following commands in your terminal:

```bash
git clone https://github.com/btanmay1/cascade-ai-frontend
cd cascade-ai-frontend
npm install
npm run dev
```
